# JavaScript Multiple-Choice Quiz Game

This project is a simple *MCQ (Multiple-Choice) Quiz Game* built entirely using *JavaScript*.  
The game runs inside the *browser console* and does not require any HTML or CSS.  
All the questions, options, and correct answers are stored in an array of objects.
